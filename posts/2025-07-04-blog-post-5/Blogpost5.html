## Introduction

This week, we worked on cleaning and analyzing data related to school-based harassment and bullying allegations across the United States. Our goal was to normalize the data by school enrollment in order to fairly compare states, and then create a series of maps to visually represent allegation trends by category.

We also used this opportunity to build reusable code for generating category-specific maps for sex-based, race-based, religion-based, orientation-based, and disability-based harassment.

Mapping Allegation Rates by State

We created a function to generate a choropleth map for any individual allegation category, normalized per 100 students.

plot_allegation_map <- function(allegation_column, title_label, legend_label) {
  allegation_by_type <- allegations %>%
    mutate(!!allegation_column := ifelse(.data[[allegation_column]] < 0, 0, .data[[allegation_column]])) %>%
    group_by(SCHID) %>%
    summarise(
      Allegations = sum(.data[[allegation_column]], na.rm = TRUE),
      .groups = "drop"
    )

  merged_type <- enrollment_summary %>%
    left_join(allegation_by_type, by = "SCHID") %>%
    mutate(
      Allegations = replace_na(Allegations, 0),
      Allegations_per_100_Students = (Allegations / Total_Students) * 100
    )

  state_summary <- merged_type %>%
    group_by(LEA_STATE_NAME) %>%
    summarise(
      Total_Allegations = sum(Allegations, na.rm = TRUE),
      Total_Students = sum(Total_Students, na.rm = TRUE),
      Allegations_per_100_Students = (Total_Allegations / Total_Students) * 100,
      .groups = "drop"
    ) %>%
    mutate(region = tolower(LEA_STATE_NAME))

  all_states <- unique(map_data("state")$region)

  state_map_data_complete <- tibble(region = all_states) %>%
    left_join(state_summary, by = "region") %>%
    mutate(
      Allegations_per_100_Students = replace_na(Allegations_per_100_Students, 0)
    )

  map_df <- map_data("state") %>%
    left_join(state_map_data_complete, by = "region")

  ggplot(map_df, aes(long, lat, group = group, fill = Allegations_per_100_Students)) +
    geom_polygon(color = "white") +
    coord_fixed(1.3) +
    scale_fill_viridis_c(option = "plasma", na.value = "grey90") +
    labs(
      title = title_label,
      fill = legend_label
    ) +
    theme_minimal()
}

Trends and Observations

Several regional trends emerged from the maps:

  • Massachusetts consistently showed the highest rates of allegations across all categories when normalized by student population.
  • States in the Northwest (e.g., Montana, Idaho, Wyoming) generally had very low or zero reports.
  • Religion-based harassment had much lower normalized rates across the board compared to other types of harassment.

These patterns raise important questions about whether states with high rates simply have better reporting systems, or if they genuinely experience more incidents.

Focusing on Massachusetts

Based on our analysis, we plan to focus on Massachusetts in future work. This state stands out in every category of normalized allegations, making it a useful case study for deeper analysis.

Our next steps will include:

  • Investigating specific schools in Massachusetts with the highest allegation rates.
  • Looking at urban vs. rural trends within the state.
  • Exploring whether school-level resources or demographics explain higher incident rates.

Questions for Further Exploration

As we move forward, we want to explore:

  • Which specific schools in Massachusetts are driving the high allegation rates?
  • How do factors like student population, school funding, or counselor availability correlate with these allegations?
  • Can we use demographic breakdowns to understand if certain student groups are overrepresented in the data?
  • Do schools in districts with high support resources report fewer incidents?

We believe narrowing our analysis to a single state will help us identify actionable patterns and inequalities more clearly.
